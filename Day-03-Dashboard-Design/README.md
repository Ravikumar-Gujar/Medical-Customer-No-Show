# Day 3 – Dashboard Design
# 📊 Blinkit Sales Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes Blinkit's sales performance using multiple datasets. It provides insights into orders, revenue, product categories, payment methods, delivery status, and monthly sales trends.

## 📈 Dashboard Features
- Total Orders KPI
- Total Revenue KPI
- Average Order Value
- Average Delivery Time
- Revenue by Category
- Top Selling Categories
- Payment Method Distribution
- Monthly Sales Trend
- Delivery Status Distribution

## 🛠 Tools Used
- Power BI
- DAX
- Power Query
- CSV Dataset
