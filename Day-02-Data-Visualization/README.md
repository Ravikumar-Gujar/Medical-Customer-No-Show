### Page 3 – Insights

Key findings from the dashboard.

---

## 📊 Key Insights

- Total Appointments: **110,527**
- No-Show Patients: **22,319**
- No-Show Rate: **20.2%**
- Female patients had more no-shows than male patients.
- Patients aged **0–18** recorded the highest number of no-shows.
- Patients without scholarships accounted for most no-shows.
- Patients without SMS reminders had more missed appointments.
